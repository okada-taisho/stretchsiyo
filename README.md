## プライバシーポリシー

ストレッチしよ（以下、「本アプリ」）は、ユーザーのプライバシーを尊重し、以下の方針に基づき情報を取り扱います。

### 1. 取得する情報について
本アプリは、ユーザーの氏名、住所、メールアドレスなどの個人情報を取得しません。  
アプリ内に入力されたメモ内容は、ユーザーの端末内にのみ保存されます。

### 2. 利用目的
本アプリは、トレーニングの記録およびメモ機能の提供のためにのみ情報を利用します。

### 3. 第三者提供について
本アプリは、取得した情報を第三者に提供することはありません。

### 4. お問い合わせ
本ポリシーに関するお問い合わせは、上記メールアドレスまでご連絡ください。

### 5. 改定について
本ポリシーの内容は、必要に応じて変更されることがあります。
